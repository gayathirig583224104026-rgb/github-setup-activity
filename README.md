# Student Portfolio & Task Management System

## Project Description
This repository contains a mini project developed as part of the GitHub Setup Activity.

The main objective of this project is to understand version control, repository management, and file organization using GitHub.

This project simulates a simple student workspace where personal details and sample programs are maintained in an organized manner.

## Objectives
- Learn Git and GitHub basics
- Create and manage repositories
- Upload and track files using commits
- Maintain project documentation
- Understand version control workflow

## Repository Structure
README.md – Project documentation
student_info.txt – Student details
hello.py – Sample Python program

- Git
- GitHub
- Python
- VS Code / Text Editor
